《荒野大镖客2》又没有PC版：R星到底在想什么？

普利尼

二十年老炮玩家

0 发布时间：2018-10-29 16:26:43
# 药丸小讲堂 #

是什么，让PC玩家们缺席了这场开放世界的万众狂欢？

好了，上周五不请假的R星粉都该枪毙――《荒野大镖客2》都发售了你还上班？

是的，这个游戏发售几天，MC评分直接飙到了95+，全世界各大网络社区都变着八百万种花样去吹爆这款RockStar倾全社之力，历时8年制作的作品。但是小编很清楚有那么一帮人，现在可不那么开心：PC玩家。


事先声明，我特讨厌这图

你看，明明就跟“三男一狗”同一个RAGE引擎，但就是PC独不占。你说咱们PC玩家如何服气？但事实上，R星从来就是这么有性格，与其他尽可能兼顾所有平台的3A大作厂商不同。这个2K旗下的老牌工作室，哪怕赚得盆满钵满，也好像从来不怎么重视PC平台――不说别的，前作《荒野大镖客》就从来没上过PC平台，所以也难怪PC玩家们这么不安。


7万人的情愿，但不知道有没有用？

但我认为在一个资本主义社会里，唯一一个永恒的定理就是：不赚钱的公司都是沙雕公司。

所以小编深信，R星这种反常的作风，背后必定有其客观事实与商业逻辑。

所以咱们为啥不探究一下呢――好歹让咱们PC党明白，到底“输”在哪儿。

一、优化问题：真的没法做

其实老玩家都知道，在R星赖以成名的《侠盗猎车手3》时代，他们的游戏实际上都会同步推出PC平台。

直到了《GTA4》经过了大半年的移植，《GTA4》当年的PC优化情况简直可以用“灾难”来形容。从这个使用R星次时代引擎“RAGE”的第一作就能看出来，RAGE这个引擎，天生就与硬件环境复杂的PC平台相性不合。


有时候甚至要开这样的低画质

以至于到了《GTA5》，R星整整用了一年多的时间潜心移植，才让PC版达到了及格的品质――而值得一提的是，无论是GTA4还是GTA5，Rockstar都是以一个工作室开发，另一个工作室移植的方式去进行双线操作的。

所以我们再来看看《荒野大镖客2》：先不说前作班底RockStar San Diego根本就不怎么会做PC游戏。甚至明说了“系列不会登陆PC”这样的狠话；咱们就客观地想想――这么个倾Rockstar八大工作室所有力量开发的游戏。哪里还来人力用于移植PC？

光是游戏的制作本身，以及现在发售之后的修修补补，就已经足够让R星全体同仁精疲力竭――相信任何一个玩过本作的朋友都不会怀疑这一点：从这个游戏的任何一个细节，每一处风景里，咱们看到的都是绿油油的美钞，与大批设计师们天天通宵的“肝”。


每一个像素都是美术与程序员的血泪

从这个角度看，RockStar真的是一个工匠精神非常重的游戏公司，现在应该已经没有几个厂商跟他们一样。将除宣传外的所有资源都投入到游戏制作本身――但现在看来，在花精力去做PC移植与更专注与游戏本身上，R星还是选择了后者。

二、经济问题：真的没钱赚

当然了，这不代表着他们就是一朵白莲花――R星之所以在上述选择中不青睐PC，还有一个非常重要，LOW比，且赤裸裸的商业原因：PC不够赚钱啊老铁。

你应该会觉得：好了小编，你又在睁眼说瞎话了。在STEAM平台创造了1000万销量的《GTA5》岂会不够赚钱？但先不说这1000万分里有多少打折、低价区（比如咱们）的水分在里面，今年5月份《GTA5》的总销量已经突破9500万了，现在说不定已经突破一亿。这1000万在这里占了多少比例，咱们心知肚明。


一千万的确很多，但对《GTA5》，还不够多

再看看之前的《GTA4》，在销量“占卜网”VGChartz上，我们能看到该作的总销量也高达2200万，但PC版却只有可怜的87万――哪怕考虑到其优化极差所带来的口碑崩坏，这对比也太可怜了。


太惨了

所以真的，在除国内以外的游戏市场，游戏主机以前是，现在是，将来应该也是绝大部分人打游戏的第一选择――更低廉的硬件成本，更简单的学习门槛。你想不出来人家老外“小白”玩家为何要选择PC。如此的销量对比也就顺理成章了。

再回到《荒野大镖客2》，这么一个前作只卖出1500万份的，市场不如《GTA5》的作品。在R星花费极大投入的前提下，在额外花出精力与人力去出PC这么个产出前景未明的平台。不得不说，实在有点风险――咱们何不像《GTA5》一样，先全力把做一波无敌的口碑，再带着“GOTY”的光环在PC平台来个“王者归来”呢？

三、Mod问题：真的不敢做

小编发现许多人将R星不热衷于发售PC版这个事儿归咎于“盗版”。

但我认为哪怕真的有盗版相关的因素，也不会是R星对PC如此谨慎的原因――他们在PC平台上吃过的鳖，可比区区盗版带来的损失严重多了。

是的，我又要说到老玩家们当年都喜闻乐见的“热咖啡”事件了：在让《GTA》系列达到巅峰的《圣安地列斯》发售后，有一个黑客在PC平台发布了一个名为“热咖啡”的补丁：说白了，就是个“你懂的”小游戏。这个现在咱们都是司空见惯的东西，在当时可是惊动了整个游戏社区。


其实现在看来真的没啥意思...

更重要的是，包括主机平台上各种蛛丝马迹表明了，这个所谓的“热咖啡”，根本就是R星在游戏设计阶段就放进去的东西，黑客做的只是在PC版将其“解锁”罢了。也就是说，这压根就是R星自己造的孽――这些可好了，“黄赌毒”你GTA全占齐了。

R星这可百口莫辩了，他们花了整整2500美元去回收游戏并修改ESRB评级。GTA系列也在这一次风波中，彻底成为了反对游戏的社会人士（比如咱们很熟悉的希拉里・克林顿）眼里，最好的头号靶子。咱们回过头来看看这个事件，你就会发现：PC平台的自由解包与自制MOD的泛滥，就是弄得R星满头是包的罪魁祸首。

所以从这一役开始，R星就对PC平台上的社区自制内容极为警惕，在《GTA5》时更是有一段时间直接封杀了最著名的MOD工具OpenIV。弄得MODDB等游戏社区满城风雨。最后还是以R星自己妥协收场。


但玩过PC版《GTA5》的同学都知道，MOD破坏力最强的，其实还不是另一个“热咖啡”，而是那极难防备的作弊外挂。而这次R星妥协的后果就是――PC版《GTAOL》几乎被废了一半。如果你们现在有兴趣打开一下Steam的《GTAOL》，你会在几乎任意一个公共房间里欣赏到各路“神仙”在疯狂斗法――当然，前提是他们现在还有兴趣继续在里面“斗法”的话。


比如这种

这就是MOD，它们是PC平台最有趣的玩意儿，但同时也是厂商最大的不可控因素：尤其是RockStar系游戏那过于与现实贴近的题材，如果配上完全失控的社区自制内容。简直就等于将自己的游戏架在火上烤――就拿《荒野大镖客2》举例，光是那些以杰克为首的各种小孩子建模，就足以让某些4ch的禽兽能想出一千种可怕的MOD。你到时候让R星怎么办？

管也是没法管的，你看隔壁贝塞斯达。经营了这么久的官方MOD社区结果不还是一团糟？所以更好的办法或许就是：先不出PC了，起码也得等盯上游戏的那些不怀好意的“眼睛”散得差不多再说。

写在最后

在这两天，也有网友通过对《荒野大镖客2》APP的解包，发现了PC平台相关的痕迹：从这一点上，也多多少少看出，PC平台的推出，起码也是在R星工作室的议事版上的。


有希望？

但这个其实也不代表我们就一定能在短期内看到这个消息。事实上，如果集合小编上述的几点因素。你突然就会发现，R星这还愿意去做PC版这样“吃力不讨好”的活儿，的确算是真爱这个平台了。

所以，如果你是真的那么想玩《荒野大镖客2》，为什么暂时不去想一些更现实的办法：比如去买一台游戏机？


起码在PC平台你没法这么玩儿
