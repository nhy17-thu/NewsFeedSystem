英伟达展示RTX光线追踪Demo 重现阿波罗11登月

作者： 你真是天才咸鱼 0 发布时间：2018-10-13 18:10:38

新一代的英伟达RTX显卡最大的特点就在于其光线实时追踪功能，在10月10日的GTC大会上英伟达公布了一段Demo演示，展示了阿波罗11登月的场景。


今天英伟达的官方油管频道也放出了完整的演示，无论是宇航员的头盔面罩，登月舱的反射还是整体光影效果都显示出了光线追踪所带来的光影提升。

Lunar Landing_ NVIDIA RTX Real-Time Ray Tracing Demo （来源：网易）
