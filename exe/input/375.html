PSN账号改名即将上线，何不找大湿我给你算一卦？

# 药丸小讲堂 #

改名啦改名啦，金木水火土，急急如律令，让老夫给你摸骨算名~

都说取名字是头等大事，这一条法则在游戏界依然适用。

名字取好了自己看着开心别人看着也高兴，要是胡乱取得你每天上线都得先被尴尬一脸。若现在还在用几年前的过时昵称或一串英文字符，还免不了遭到其他玩家冷嘲热讽。

这时候该怎么办？改名啊！现在的游戏早把改名卡摆上商城，掏点琐碎银子就能“重新做人”。

但是，对于索尼游戏爱好者来说，他们最近才姗姗来迟的收到官方正在筹备改名许可的通知。

这一则消息对PSN用户来说真是等得太久了，对老粉丝来说，他头上顶着的可能是10年前的中二回忆，指不定还有关于前前前任的惨痛回忆。如今骚尼粑粑终于网开一面允许玩家更改ID，这不得好好琢磨一下？

实不相瞒，老夫浸淫改名文化多年，早已参透QQ网名、非主流小情侣爱称等取名宝典。你既然点进我的文章那必然是与我有缘，也就略微给你指点一二吧。


排好队，一个一个来

够用就好型

名字这东西，实际生活中的最大作用就是让人记住，无需多复杂，人海茫茫能记住唯一那个你就够了。比如金庸笔下的武侠小说主角人名就大多憨厚朴实，郭靖、黄蓉、杨过、乔峰、虚竹。不需要从辞海中翻找多么高深的词汇，亦不用绞尽脑汁去想什么刻骨铭心的寓意，就挑几个简单容易又好记的，作为一个符号完全够用。

不好理解是吧？咱先看几个个例。


闯荡江湖多年的小虾米，脱胎自国产经典游戏《金庸群侠传》。虽然武功平平，但天资聪敏又刻苦勤学，机缘巧合之下和多个江湖门派结缘，并收得多个失传已久的武功秘籍，潜心研究后更受高人点拨。最终成长为名震江湖的一代英豪，成为武林盟主。

姓名中的“虾米”二字，是池塘里最不受待见的生物链底端生物，最为卑劣，没几个人愿意担着这名号走江湖。所以不常见，但更易被人记住。对比他日后的丰功伟绩，更为“小虾米”增色不少。


你看这位仁兄的名字，短短五个字深刻的表露了他个人最大的特征――鸡儿，短。

谁的？他的。都说一个男同胞那得顶天立地，哪怕退一步，就算不是晚上生龙活虎，至少也得早起一柱擎天。像“哥”这样生理天生有缺陷的同志，光是听过一次名字就能让人永远铭记，日常使用绰绰有余尔。


守望玩家在游戏结算阶段有点赞系统，点赞最多的玩家会获得高亮特效，并且还有额外经验加成。于是赛后点赞就成了不少玩家竞相追捧的香饽饽。

你看，这位我素未谋面的野生儿子从名字上就满是对为父的尊敬，你说看见这么乖巧的后生晚辈我不得赏他一个大的的赞？走你~


儿砸休走，爸爸给你一个大大的赞


别看这位小改改只露了半身像（对A，要不起），从她长达8个字的昵称中我们不难窥探到她最引以为傲的身体曲线。能在国服称第一的担当？还特么是人类盆骨部分后方的浑圆部位，这真是太赤鸡了呀。


啊~我喜欢！


俗话说男女搭配干活不累，没有妹纸的汉子是孤独寂寞冷的，在大雨漂泊风起云涌狂风暴雨中只能一人独自在冷风中瑟瑟发抖。

不过这位选手不一样，人家靠的是一双手，何须女朋友？名字正是印证了一句老话――男儿当自强。


瞧这师徒五人，取自我国经典文学名著《西游记》，看上第一眼脑海里就回响起熟悉的BGM，妖怪哪里跑？


以小队组合形式出道的话，最适合的自然是互为补足的团队名。上图6位相貌甚是统一的大妹子单看模样真让人傻傻分不清楚。加上所属部门头衔就清晰明了易于分辨了，从神经内科到消化内科，整容大夫到呼吸内科一条龙服务到位。

诸位姐姐，请给我做全套好吗！


国外玩家已经组建好了全新团队

可是你说吧，既然是要取名，如果太过四平八稳是否又有些对不起狂拽酷炫的自己？这姓名毕竟也是给人的第一印象，不突出一点就很容易淹没在人来人往。莫慌，莫急，大湿我这就给你几个彰显个性的昵称作参考。

下面这是90后熟悉的“火星体”，曾一度占据网络昵称大壁江山，甚至在刚兴起之初还受到网络其他用户的鄙夷之色。如今火星语言演化至今，已经成为独树一帜的网络文化，充满个性的人士都愿意选上一符半字表现自个儿放荡不羁的青春。


这第一个是最爱你的俄，第二个是….唔再下一个，第三个是援你，也浪你，第四个是……看不懂。

看来是这届昵称不行，让老湿换一届。


看看，看看！上述昵称无一不是万里挑一的臻美之词。想一想新武侠主义的创新者古龙先生。他笔下的人名那个不都是超凡脱俗不落俗套，仿佛高山流水诗情画意，又不失典雅亲近，叫人拍手称快。


推荐你啪嗒砰同款文字

上述这些个字眼与古龙先生生花妙笔下的角色名相比起来也不逞多让，读起来飘忽悠悠，有点不食人间烟火的美妙，倘若选上一个必能在万人群中吸引住眼球。


像是走个复古风来上一串象形字更能增添几分姿色

无论在何时何地，能九转回肠吊人胃口的好名字从来都不会遭到抛弃，像“葬爱清香蓝爵”、“东京樱花雨打湿巴黎的街”等麻酥酥的的昵称都可以做备选，回头率没得说，听老湿的准没错。

PSN ID

最后说回咱们的PSN账号名，索尼对账户名的创建有着严格的要求，最少三个字符、最多16个字符，可包含字母、数字等符号，并不支持中文。所以…是的，前面的汉语昵称只是演习，真正的取名时刻从现在开始。


若是想要自己记住，别人也一目了然，那最好从自己熟悉的事物下手琢磨。譬如QQ号、手机号、网名和手机号末尾。选择这一路子别人方便你也方便，网络遇上了加个好友也简单。

据上面这位兄弟痛彻心扉的讲述，他用这ID好多年，节操掉了遍，基友成群结队。


从喜欢的游戏中找灵感更容易，也可以引起别的玩家共鸣。上图这位“last-of-bus”就是取自“the last of us”，索尼独占大作《最后的生还者》英文原名。这可是有口皆碑的神作，取了这串名号走到哪都不缺组织。

或者也有“Super noob is me”（我是超级菜鸟）、“Big dick”、“Little brother”等含义隐晦的名字，基本都是乍看之下普普通通，关键时刻会让人会心一笑的命名，按需取用。


其实昵称这东西，取得时候未必会想太多，如今的网络游戏、网络社区几乎都会考虑到玩家有更换名称的需求。就好比一件衣服穿旧了会想换一件，过去的事儿时间久了也该翻个篇，户口本上的姓名都能改，何况虚拟世界的称谓。

只是最值得吐槽的是，PSN在2006年上线，“改名”这一玩家们的刚性需求索尼硬是头铁到2018年底才开始测试，隔壁Xbox和暴雪战网也早在好几年前就开启了类似服务。

当然，改名怎么说也不是随便拍个脑门就能通过的，所以大部分游戏昵称都有更改次数的限制或是有偿更改。目前索尼计划PSN ID首次修改免费，之后9.99美元/次，PS Plus可享受4.99美元/次的会员价，与Xbox Live的策略别无二致（不同服务器政策不同，实际价格看实所在区服）。


最后大湿我再给各位即将重新选择姓名的PlayStation用户一点建议，昵称虽然可以说是完全私人化的信息，不过也得考虑下他人的看法，取个自己叫着舒心别人听着顺心的就好。

像之前国外就有很多不守规矩的用户恶意在昵称中隐藏不良信息，结果惨遭官方清洗，强制赠送改名券，更有甚者直接封号处理，如前段时间闹得沸沸扬扬的用户名为“Ligma”用户就惨遭天降正义（不要去查这个单词是什么意思，很污）。

希望老湿的这篇PSN取名易能多少帮助到各位，倘若各位真是绞尽脑汁也想不出好的PSN ID，那我就最后再赠予你一个压箱底的： Nintendo is NO.1


拿好，不谢
